##  Summary

**Each night, you learn which character died by execution today.**

The Undertaker learns which character was executed today.

* The player must have died from execution for the Undertaker to learn who they are.
  Deaths during the day for other reasons, such as the [Gunslinger](/en/characters/travellers/gunslinger) choosing a player to kill,
  or the exile of a Traveller, do not count.
* The Undertaker wakes each night except the first, as there have been no executions yet.
* If nobody died today, the Undertaker learns nothing.
  The Storyteller either does not wake the Undertaker at night,
  or wakes them but does not show a card.
* If the [Drunk](/en/characters/outsiders/drunk) is executed,
  the Undertaker is shown the [Drunk](/en/characters/outsiders/drunk) character token,
  not the token for the Townsfolk that the Drunk player thought they were.

##  How to Run

If a player dies by execution,
put the Undertaker's **DIED TODAY** reminder token by the dead player's character card.

Each night except the first, if any player died by execution today, wake the Undertaker.
Show the character card marked **DIED TODAY** to the Undertaker.
Put the Undertaker to sleep.
emove the Undertaker's reminder token when convenient.

In [Trouble Brewing](/en/scenarios/trouble-brewing), there can only be one execution per day,
and every execution causes a player to die.
In other editions, there may be more than one execution per day
(in which case the Storyteller chooses which character to show the Undertaker)
or the execution does not cause a death (in which case the Undertaker learns nothing).

##  Examples

{{< example >}}
The [Mayor](/en/characters/townsfolk/mayor) is executed today.
That night, the Undertaker is shown the Mayor card.
{{< /example >}}

{{< example >}}
The [Drunk](/en/characters/outsiders/drunk),
who thinks they are the [Virgin](/en/characters/townsfolk/virgin), is executed today.
At night, the Undertaker is shown the [Drunk](/en/characters/outsiders) card,
because the Undertaker learns a player's true character,
as opposed to the one they believe they are.
{{< /example >}}

{{< example >}}
The [Spy](/en/characters/minions/spy) is executed.
Two Travellers are exiled.
That night, the Undertaker is shown the [Butler](/en/characters//outsiders/butler) card,
because the [Spy](/en/characters/minions/spy) is registering as the [Butler](/en/characters/outsiders/butler),
and because the exiles are not executions.
{{< /example >}}

{{< example >}}
Nobody was executed today. That night, the Undertaker does not wake.
{{< /example >}}
