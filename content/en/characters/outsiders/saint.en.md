##  Summary

**If you die by execution, your team loses.**

The **Saint** ends the game if they are executed.

* If the **Saint** dies by execution, the game ends. **Saint**'s team wins.
* If the **Saint** dies in any way other than execution 
  _(such as the Demon killing them)_ then the game continues.

##  How to Run

If the **Saint** dies by execution, declare that the game ends and evil wins.

The **Saint**'s ability has the strange phrase _"your team loses."_  
In [Trouble Brewing](/en/scenarios/trouble-brewing), this will always mean _"evil wins"_
because the **Saint** is always _good_.  
In other editions, it is possible for characters to change alignment.
If the _evil Saint_ is executed, good wins.

##  Examples

{{< example >}}
There are seven players alive and nominations are in progress.
The **Saint** gets four votes and is about to _die_.
Then, the [Baron](/en/characters/townsfolk/baron) is nominated but only gets three votes.
No more nominations occur today.
The Saint is executed, and _evil_ wins.
{{< /example >}}

{{< example >}}
The [Imp](/en/characters/demons/imp) is nominated, and the players vote.
The [Gunslinger](/en/characters/travellers/gunslinger) kills the **Saint**.
The **Saint** dies, and the game continues.
{{< /example >}}

{{< example >}}
The **Saint** is _executed_.
However, the [Scapegoat](/en/characters/travellers/scapegoat)s ability is triggered,
so the [Scapegoat](/en/characters/travellers/scapegoat) dies instead.
The game continues, because the **Saint** did not die.
{{< /example >}}
