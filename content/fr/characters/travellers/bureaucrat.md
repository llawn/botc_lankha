## Résumé

**Chaque nuit, choisissez un joueur (pas vous-même) : son vote compte pour 3 votes demain.**

Le Bureaucrate donne des votes supplémentaires à un joueur de son choix.

* Quand un joueur choisi par le Bureaucrate vote, ce vote compte pour trois votes. Cela se produit chaque fois que ce joueur vote ce jour-là.
* Le joueur avec le triple vote le perd immédiatement si le Bureaucrate meurt, y compris si le Bureaucrate est exilé, car le Bureaucrate perd sa capacité.
* Les exils ne sont jamais affectés par les capacités, donc le joueur avec le triple vote ne peut soutenir les exils qu'une fois, pas trois fois.
* Puisque le Conteur compte le nombre de votes à haute voix en déplaçant sa main autour du cercle, tous les joueurs sauront quel joueur le Bureaucrate a choisi.

## Comment animer

Chaque nuit, réveillez le Bureaucrate. Il désigne un joueur du doigt. Marquez le joueur choisi avec le rappel **3 VOTES** du Bureaucrate. Rendormez le Bureaucrate.

Chaque fois que vous comptez le vote d'un joueur marqué **3 VOTES**, comptez-le comme trois votes au lieu d'un. (Comptez à haute voix, comme d'habitude.)

## Citation

*Signez ici, s'il vous plaît. Et ici. Et ici. Eeeeeet ici. Tout devrait être classé et compté d'ici la fin de la journée, en supposant que les signatures de tout le monde soient lisibles. Cela fait des lustres qu'il n'y a pas eu d'erreur dans la paperasse. Hier midi, si ma mémoire est bonne...*

## Exemples

{{< example >}}
Le Bureaucrate choisit Evin. Le lendemain, quand le premier vote est compté, le Conteur compte « 1... 2... 3... 4-5-6... 7 ». Le joueur nommé a maintenant sept votes pour son exécution, et le processus de nomination continue.
{{< /example >}}

{{< example >}}
Le Bureaucrate choisit Filip. Le lendemain, Filip a un triple vote, qu'il utilise pendant quatre nominations.
{{< /example >}}

{{< example >}}
Le Bureaucrate choisit Douglas, qui est mort. Le lendemain, Douglas utilise son jeton de vote pour voter, et son vote compte triple.
{{< /example >}}
