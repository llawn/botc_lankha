## Résumé

**Chaque nuit, choisissez un joueur (pas vous-même) : demain, vous ne pouvez voter que si ce joueur vote aussi.**

Le Majordome ne peut voter que lorsque son Maître (un autre joueur) vote.

* Chaque nuit, le Majordome choisit un joueur pour être son Maître. Cela peut être le même joueur que la nuit précédente ou un différent.
* Si le Maître a levé la main pour voter, ou si le vote du Maître a déjà été compté, le Majordome peut lever la main pour voter.
* Si le Maître a baissé la main, signalant qu'il ne vote pas, ou si le Maître baisse la main avant que son vote ne soit compté, le Majordome doit baisser la main également.
* Ce n'est pas la responsabilité du Conteur de surveiller le Majordome. Il est responsable de son propre vote. Voter délibérément quand il ne le devrait pas est considéré comme de la triche.
* Comme les exils ne sont jamais affectés par les capacités, le Majordome peut voter librement pour un exil.
* Les joueurs morts ne peuvent lever la main pour voter que s'ils ont un jeton de vote. Si le Majordome choisit un joueur mort comme Maître, cela s'applique toujours.
* Le Majordome n'est jamais forcé de voter.
* Le vote du Majordome peut être compté par le Conteur avant ou après celui de son Maître. La position autour de la table n'a pas d'importance.

## Comment animer

Chaque nuit, réveillez le Majordome. Il désigne un joueur. Marquez ce joueur avec le jeton de rappel **MAÎTRE**. Rendormez le Majordome.

Durant une nomination, le Majordome ne peut avoir la main levée pour voter que si le Maître a la main levée pour voter ou si le vote du Maître a déjà été compté.

Si le Majordome vote accidentellement illégalement, comptez quand même son vote. Si vous ne le faites pas, le total sera inférieur à ce qu'il devrait être, et tous les joueurs sauront quel joueur est le Majordome, ce qui est injuste pour l'équipe maléfique. Les erreurs arrivent. Continuez à jouez et discutez-en en privé plus tard.

## Citation

*Oui, monsieur... Non, monsieur... Certainement, monsieur.*

## Exemples

{{< example >}}
Le Majordome choisit Filip comme Maître. Demain, si Filip lève la main pour voter lors d'une exécution, alors le Majordome peut aussi le faire. Sinon, le Majordome ne peut pas lever la main.
{{< /example >}}

{{< example >}}
Une nomination est en cours. Le Majordome et son Maître ont tous deux levé la main pour voter. Alors que le Conteur compte les votes, le Maître baisse la main à la dernière seconde. Le Majordome doit baisser la main immédiatement.
{{< /example >}}

{{< example >}}
Le Majordome est mort. Parce que les joueurs morts n'ont pas de capacité, le Majordome peut voter avec son jeton de vote à tout moment.
{{< /example >}}
