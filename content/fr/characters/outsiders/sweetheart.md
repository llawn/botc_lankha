## Résumé

**Quand vous mourez, 1 joueur est ivre à partir de maintenant.**

Le Bien-Aimé, quand il meurt, rend quelqu'un ivre pour le reste de la partie.

* Le Conteur choisit quel joueur devient ivre.
* Cette capacité fonctionne tant que le Bien-Aimé est mort.

## Comment animer

Si le Bien-Aimé meurt, vous choisissez n'importe quel joueur pour devenir **ivre** — marquez ce joueur avec le rappel **IVRE**. Le Bien-Aimé ne perd pas cette capacité en mourant.

Vous voudrez presque toujours rendre un Villageois ivre, ce qui nuira au camp bon, mais si le camp maléfique gagne de façon assez convaincante, vous pouvez rendre un Outsider, un Sbire, ou même le Démon ivre.

## Exemples

{{< example >}}
Le Bien-Aimé meurt. Le Mathématicien est maintenant ivre, et peut recevoir de fausses informations la nuit.
{{< /example >}}

{{< example >}}
Le Bien-Aimé meurt. Le Mutant est maintenant ivre. Le Mutant peut se révéler être le Mutant en toute sécurité, mais il ne le sait pas.
{{< /example >}}

{{< example >}}
Le Bien-Aimé meurt. Le Démon est maintenant ivre, donc son attaque de nuit ne tuera personne.
{{< /example >}}
