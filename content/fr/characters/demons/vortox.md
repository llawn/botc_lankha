## Résumé

**Chaque nuit*, choisissez un joueur : il meurt. Les capacités des Villageois donnent de fausses informations. Chaque jour, si personne n'est exécuté, les maléfiques gagnent.**

Le Vortox rend toutes les informations fausses.
* Chaque fois qu'un Villageois reçoit des informations de sa capacité, il reçoit de fausses informations. Même s'il est ivre ou empoisonné, elles doivent être fausses.
* Le Vortox n'affecte pas les informations obtenues par d'autres moyens, comme lorsque le Conteur explique les règles, ou lorsque le personnage ou l'alignement d'un joueur change.
* Quand la nuit tombe, si personne n'a été exécuté aujourd'hui, les maléfiques gagnent. Exiler un Voyageur ne compte pas.

## Comment animer

Tant que le Vortox est vivant, vous devez donner de fausses informations chaque fois qu'une capacité de Villageois vous demande de donner des informations.

Chaque nuit sauf la première, réveillez le Vortox. Il montre un joueur. Ce joueur **meurt** — marquez-le avec le rappel **MORT**. Rendormez le Vortox.

Chaque crépuscule, si aucun joueur n'a été exécuté aujourd'hui, la partie se termine et l'équipe maléfique gagne.

## Citation

*« Noir est Blanc. Bien est Mal. Gauche est Droite. Haut est Long. Bas est Vue. Court est Aveugle. Suivez-moi. Réponses trouvées. »*

## Exemples

{{< example >}}
Le Vortox tue le Sage. Le Sage apprend deux joueurs, dont aucun n'est un Démon.
{{< /example >}}

{{< example >}}
Personne n'a voté ou nominé aujourd'hui, mais le Mutant est exécuté. Cette nuit, la Fleuriste et le Crieur Public apprennent tous deux un « oui ».
{{< /example >}}

{{< example >}}
Le Savant est en jeu et apprend deux informations chaque jour. Les deux sont fausses. Cette nuit, le Rêveur choisit un joueur qui est le Savant et apprend que ce joueur est soit le Philosophe, soit le No Dashii.
{{< /example >}}

{{< example >}}
Le Pit-Hag transforme le Jongleur en Sorcière. Le Jongleur apprend qu'il est maintenant la bonne Sorcière, car cette information vient de la capacité du Pit-Hag, pas d'une capacité de Villageois.
{{< /example >}}

{{< example >}}
Aujourd'hui, un joueur est mort de la Sorcière, deux Voyageurs ont été exilés, 5 nominations ont eu lieu, mais personne n'a été exécuté. Les maléfiques gagnent.
{{< /example >}}
