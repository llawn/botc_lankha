## Résumé

**Chaque nuit*, choisissez un joueur : il meurt. Vous commencez par choisir un joueur : il est empoisonné. Vous mourez si et seulement s'il est mort.**

Le Lleech vit si son hôte vit et meurt si son hôte meurt.
* La première nuit, le Lleech choisit un joueur, qui est empoisonné pour le reste de la partie.
* Si ce joueur est vivant, le Lleech ne peut pas mourir. Si le Lleech est exécuté, le Conteur dit au groupe que le joueur vit, mais pas pourquoi.
* Si le joueur que le Lleech a choisi meurt, le Lleech meurt aussi. Si cela signifie qu'il ne reste qu'un ou deux joueurs vivants, le camp du bien gagne quand même, car le Démon est mort.
* À partir de la deuxième nuit, les joueurs que le Lleech attaque meurent mais ne sont pas empoisonnés.
* Si un Lleech est créé en cours de partie, il empoisonne un joueur cette nuit-là. Il doit choisir un joueur vivant.

## Comment animer

Pendant la première nuit, réveillez le Lleech. Il montre un joueur. Ce joueur est **empoisonné** — marquez-le avec le rappel **EMPOISONNÉ**. Rendormez le Lleech.

Chaque nuit sauf la première, réveillez le Lleech. Il montre un joueur. Ce joueur **meurt** — marquez-le avec le rappel **MORT**. Rendormez le Lleech.

Si le Lleech devrait mourir mais que le joueur marqué avec le rappel **EMPOISONNÉ** du Lleech est vivant, le Lleech ne meurt pas. Si le joueur marqué avec le rappel **EMPOISONNÉ** du Lleech **est mort**, le Lleech **meurt** et le camp du bien gagne.

Lorsque vous donnez de fausses informations à un bon joueur empoisonné par le Lleech, assurez-vous que ces informations semblent vraies. Si le bon joueur croit qu'elles sont fausses, il se fera rapidement exécuter et gagnera la partie.

## Exemples

{{< example >}}
Le Lleech empoisonne le Noble. Le Noble apprend de fausses informations. Le Lleech est exécuté, mais ne meurt pas. Le lendemain, le Noble est exécuté. Le Noble et le Lleech meurent. Le camp du bien gagne.
{{< /example >}}

{{< example >}}
Le Lleech empoisonne le Fermier. Le Courtisan rend le Lleech ivre. Le Fermier empoisonné meurt, et la partie continue car le Lleech est également ivre. Le Lleech ivre est exécuté et meurt, et le camp du bien gagne.
{{< /example >}}
