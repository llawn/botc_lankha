## Résumé

**Une fois par partie, de nuit, choisissez un personnage gentil : obtenez son pouvoir. Si ce personnage est en jeu, il est ivre.**

Le Philosophe n'a aucun pouvoir jusqu'à ce qu'il décide quel personnage il veut imiter.

* Il peut le faire une fois par partie. Quand il le fait, il obtient le pouvoir de ce personnage. Il ne devient pas ce personnage.
* Il voudra peut-être attendre un moment avant de choisir. Si le Philosophe choisit un personnage déjà en jeu, le joueur de ce personnage devient ivre. Si le Philosophe meurt ou devient ivre ou empoisonné, le joueur qu'il rendait ivre redevient sobre.
* Si le Philosophe a choisi un personnage qui n'était pas en jeu à ce moment-là mais qui est maintenant en jeu, ce personnage est ivre.
* Si le Philosophe obtient un pouvoir qui fonctionne de nuit, il se réveille quand ce personnage se réveillerait. Si ce pouvoir n'est utilisé que la première nuit, il l'utilise cette nuit.
* Si le Philosophe récupère son pouvoir via le Collecteur d'Os, ou utilise son pouvoir deux fois via le Barista, le Philosophe peut choisir un nouveau pouvoir, ou le même pouvoir qu'avant.
* Si le pouvoir du Philosophe fonctionne quand il est mort, comme celui du Klutz, il fonctionne si le Philosophe est mort.

## Comment animer

Chaque nuit, réveillez le Philosophe. Il secoue la tête ou désigne une icône de Villageois ou d'Étranger sur sa fiche de personnage. Rendormez le Philosophe.

S'il désigne l'icône d'un personnage qui n'est pas en jeu, échangez le jeton du Philosophe avec le jeton du personnage choisi et marquez-le avec le jeton rappel **EST LE PHILOSOPHE**.

S'il désigne l'icône d'un personnage en jeu, le joueur du personnage choisi devient **ivre** — marquez-le avec le jeton rappel **IVRE**. (*Vous pouvez maintenant utiliser les rappels du personnage dupliqué pour le Philosophe*). Si le Philosophe meurt, le joueur rendu ivre par le Philosophe redevient **sobre** — retirez le jeton rappel **IVRE**.

## Citation

*Si quoi que ce soit est réel, la bière est réelle. Buvez, car demain nous mourrons peut-être.*

## Exemples

{{< example >}}
Pendant la première nuit, le Philosophe choisit d'obtenir le pouvoir du Rêveur. Il obtient le pouvoir du Rêveur à partir de maintenant et agit quand le Rêveur agirait normalement.
{{< /example >}}

{{< example >}}
Pendant la troisième nuit, le Philosophe choisit d'obtenir le pouvoir de l'Horloger. Cette nuit-là, il apprend la distance entre le Démon et son Sbire le plus proche.
{{< /example >}}

{{< example >}}
Un Artiste est en jeu. Le Philosophe choisit d'obtenir le pouvoir de l'Artiste. L'Artiste d'origine devient ivre. Plus tard, le Philosophe meurt, donc l'Artiste d'origine redevient sobre. (L'Artiste d'origine redeviendrait également sobre si le Philosophe devenait ivre.)
{{< /example >}}
