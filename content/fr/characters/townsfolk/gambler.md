## Résumé

**Chaque nuit*, choisissez un joueur et devinez son personnage : si vous vous trompez, vous mourez.**

Le Joueur peut deviner qui est qui... mais paie le prix ultime s'il se trompe.

* Chaque nuit sauf la première, le Joueur choisit un joueur et devine son personnage en pointant son icône sur la feuille de personnages. Si la devinette est correcte, rien ne se passe. Si la devinette est incorrecte, le Joueur meurt.
* Le Joueur n'apprend pas du Conteur si sa devinette est correcte ou incorrecte.
* Le Joueur peut choisir n'importe quel joueur, mort ou vivant, même lui-même.

## Comment animer

Chaque nuit sauf la première, réveillez le Joueur. Il désigne n'importe quel joueur, puis désigne n'importe quelle icône de personnage sur sa feuille de personnages. Rendormez le Joueur. Si le joueur choisi est un personnage différent de l'icône de personnage choisie, le Joueur **meurt** — marquez-le avec le rappel **MORT**.

## Citation

*Pile, je gagne. Face, tu perds.*

## Exemples

{{< example >}}
Le Joueur désigne le joueur Ménestrel, puis l'icône Ménestrel. Cette devinette est correcte, donc le Joueur reste en vie, mais il est tué par le Démon ce soir quoi qu'il en soit.
{{< /example >}}

{{< example >}}
L'Avocat du Diable bluffe en se faisant passer pour le Pacifiste. Cette nuit, le Joueur désigne le joueur Avocat du Diable, puis l'icône Pacifiste. Cette devinette est fausse, donc le Joueur meurt.
{{< /example >}}
