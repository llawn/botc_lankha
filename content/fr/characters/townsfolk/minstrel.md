## Résumé

**Quand un Sbire meurt par exécution, tous les autres joueurs (sauf les Voyageurs) sont ivres jusqu'au crépuscule du lendemain.**

Le Ménestrel rend tout le monde ivre si un Sbire meurt.

* Si un Sbire est exécuté et meurt, tous les joueurs (sauf le Ménestrel) deviennent immédiatement ivres et le restent toute la nuit et tout le jour suivant. Les Villageois, les Étrangers, les Sbires et même les Démons deviennent ivres, mais pas les Voyageurs. Cela ne se produit pas si un Sbire meurt de nuit.
* Si un Sbire mort est exécuté, le pouvoir du Ménestrel ne se déclenche pas — un personnage mort ne peut pas mourir à nouveau ! Si un Sbire est exécuté mais ne meurt pas, le pouvoir du Ménestrel ne se déclenche pas. Si le Ménestrel est ivre ou empoisonné quand un Sbire meurt par exécution, le pouvoir du Ménestrel ne se déclenche pas.

## Comment animer

Pendant la journée, si un Sbire meurt par exécution, tous les autres joueurs sauf les Voyageurs deviennent **ivres** — placez le jeton rappel **TOUT LE MONDE EST IVRE** du Ménestrel au centre du côté gauche du Grimoire.

Au crépuscule du lendemain, tous les joueurs rendus ivres par le Ménestrel redeviennent **sobres** — retirez le jeton rappel **TOUT LE MONDE EST IVRE**.

## Citation

*Et j'entendrai, bien que douce tu marches au-dessus de moi... Et tous mes rêves seront plus chauds et plus doux... Si tu n'oublies pas de me dire que tu m'aimes... Je dors simplement en paix jusqu'à ce que tu viennes à moi.*

## Exemples

{{< example >}}
Pendant le premier jour, le Pacifiste meurt. Cette nuit-là, les joueurs agissent normalement, car le Pacifiste n'est pas un Sbire. Pendant le deuxième jour, le Juge exécute le Parrain. Cette nuit-là, tout le monde est ivre, y compris le Démon, donc personne ne meurt. Pendant le troisième jour, un Sbire protégé par l'Avocat du Diable est exécuté et meurt, car l'Avocat du Diable est ivre. Encore une fois, puisqu'un Sbire est mort pendant la journée, le Ménestrel rend tout le monde ivre.
{{< /example >}}

{{< example >}}
L'Assassin est exécuté, donc le Ménestrel rend tout le monde ivre. Le lendemain, le Parrain est exécuté, donc le Ménestrel rend tout le monde ivre à nouveau. Le Démon n'a pas pu tuer ces deux nuits-là. Le lendemain, le Voyageur Apprenti Maître du Mal est exilé, donc cette nuit-là tout le monde redevient sobre, car le pouvoir du Ménestrel ne se déclenche pas quand un Voyageur est exilé.
{{< /example >}}

{{< example >}}
Pendant la journée, l'Assassin meurt, donc le Ménestrel rend tout le monde ivre. Le jour suivant, le Zombuul est exécuté et meurt pour la première fois. Les gentils gagnent, car le Zombuul est ivre et n'a donc aucun pouvoir.
{{< /example >}}
