## Résumé

**Chaque nuit, choisissez un joueur vivant : soit vous, soit il est ivre jusqu'au crépuscule. Vous ne pouvez pas mourir.**

Le Sailor est soit ivre, soit rend quelqu'un d'autre ivre. Tant que le Sailor est sobre, il ne peut pas mourir.

* Chaque nuit, le Sailor choisit un joueur, qui sera probablement ivre.
* S'il se choisit lui-même, il perd sa capacité « ne peut pas mourir » jusqu'à ce qu'il redevienne sobre.
* Si le Sailor choisit accidentellement un joueur mort, le Conteur lui demande de choisir à nouveau.
* Si le Sailor choisit un autre joueur, le Conteur décide quel joueur est ivre. S'il choisit un Villageois, le Conteur rend généralement le Villageois ivre, mais si un Outsider, un Sbire ou le Démon est choisi, le Conteur rend généralement le Sailor ivre.
* Tant qu'il est sobre, le Sailor ne peut pas mourir, même s'il ne s'est pas encore réveillé la nuit pour boire.

## Comment animer

Chaque nuit, réveillez le Sailor. Il désigne un joueur. Rendormez le Sailor. Soit le Sailor, soit le joueur choisi devient **ivre** — marquez-le avec le rappel **IVRE**.

Si le Sailor sobre devrait mourir, le Sailor reste en vie. Si le Sailor sobre est exécuté, déclarez que ce joueur est exécuté mais reste en vie. (*Ne dites pas pourquoi.*)

## Exemples

{{< example >}}
Le Sailor choisit l'Exorcist, et le Conteur décide que l'Exorcist est ivre. Cette nuit-là, le Sailor est attaqué par le Shabaloth. Le Sailor reste en vie. Le lendemain, le Sailor est exécuté mais reste en vie.
{{< /example >}}

{{< example >}}
Pendant la journée, le Gossip a fait une déclaration publique qu'il pensait fausse, mais qui était en fait vraie. Cette nuit-là, la capacité du Gossip tue un joueur. Le Sailor s'est rendu ivre lui-même, et le Conteur décide que le Sailor meurt.
{{< /example >}}

{{< example >}}
Le Sailor choisit le Mastermind, mais le Conteur décide que le Sailor est ivre. Le lendemain, le Sailor demande à être exécuté pour « prouver qu'il est le Sailor », mais meurt parce qu'il est ivre.
{{< /example >}}
