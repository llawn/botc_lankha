## Résumé

**Chaque nuit*, vous apprenez si un Démon a voté aujourd'hui.**

La Fleuriste sait si le Démon a voté ou non.

* Le vote d'un Démon compte, que le candidat ait été exécuté ou non.
* La Fleuriste ne détecte pas si le Démon a levé la main pour d'autres raisons, comme quand les joueurs « votent » pour ce qu'ils veulent manger, ou quand les joueurs lèvent la main pour exiler un Voyageur.
* Si le Démon change de joueur après que le Démon d'origine a voté mais avant que la Fleuriste ne se réveille pour apprendre ses informations, la Fleuriste détecte le Démon d'origine.
* S'il y a deux (ou plus !) Démons, même morts, la Fleuriste détecte si l'un d'eux a voté. Si même un seul Démon a voté, la Fleuriste apprend un « oui ».

## Comment animer

Chaque aube, marquez la Fleuriste avec le rappel **DÉMON N'A PAS VOTÉ**, et retirez le rappel **DÉMON A VOTÉ**, le cas échéant.

Chaque jour, si le Démon vote pour une exécution, remplacez le rappel **DÉMON N'A PAS VOTÉ** par le rappel **DÉMON A VOTÉ**.

Chaque nuit sauf la première, réveillez la Fleuriste. Si la Fleuriste est marquée **DÉMON N'A PAS VOTÉ**, secouez la tête non. Si la Fleuriste est marquée **DÉMON A VOTÉ**, hochez la tête oui. Puis, rendormez la Fleuriste.

Si vous oubliez si le Démon a voté ou non, réveillez le Démon de nuit et demandez-lui en montrant le jeton d'information **AVEZ-VOUS VOTÉ AUJOURD'HUI ?**. Il doit répondre honnêtement, puis se rendormir.

## Citation

*Les violettes d'hier se sont fanées et sont mortes, mais aujourd'hui mes coquelicots fleurissent.*

## Exemples

{{< example >}}
Il y a eu une nomination aujourd'hui. Beaucoup de joueurs ont voté, le joueur a été exécuté, mais le Démon n'a pas voté. Cette nuit, la Fleuriste apprend que le Démon n'a pas voté aujourd'hui.
{{< /example >}}

{{< example >}}
Il y a eu trois nominations aujourd'hui. Le Démon a voté lors de la deuxième nomination. Personne n'a été exécuté. Cette nuit, la Fleuriste apprend que le Démon a voté aujourd'hui.
{{< /example >}}

{{< example >}}
Il n'y a eu aucune nomination aujourd'hui. Un Voyageur a été exilé, et tous les joueurs ont levé la main pour soutenir l'exil. Cette nuit, la Fleuriste apprend que le Démon n'a pas voté aujourd'hui. (Les exils ne sont jamais affectés par les capacités.)
{{< /example >}}
