## Résumé

**Chaque nuit, choisissez un joueur vivant : un Démon choisi échange son personnage et son alignement avec vous, puis est empoisonné.**

Le Snake Charmer apprend joueur après joueur qui n'est pas le Démon... mais devient le Démon s'il devient trop gourmand ou trop audacieux.

* Chaque nuit, il choisit un joueur. Si ce joueur n'est pas le Démon, rien ne se passe. S'il est le Démon, le Snake Charmer devient ce Démon et devient méchant, et le Démon devient bon et empoisonné en permanence.
* Si le Philosopher a la capacité du Snake Charmer et devient le Démon, le Démon devient un Philosopher empoisonné.

## Comment animer

Chaque nuit, réveillez le Snake Charmer. Il désigne un joueur.

Si ce joueur n'est pas le Démon, rien ne se passe. Rendormez le Snake Charmer.

Si ce joueur est le Démon, l'ancien Snake Charmer se transforme en nouveau Démon (*méchant*), et l'ancien Démon se transforme en nouveau Snake Charmer (*bon*) — échangez le jeton du Snake Charmer et celui du Démon. Le nouveau Snake Charmer est **empoisonné** — marquez-le avec le rappel **EMPOISONNÉ**. Réveillez le nouveau Démon et montrez-lui le jeton d'info **VOUS ÊTES**, un pouce vers le bas, le jeton **VOUS ÊTES**, puis le jeton du Démon. (*Cela montre qu'il est maintenant méchant et le Démon.*) Rendormez le nouveau Démon. Réveillez le nouveau Snake Charmer et montrez-lui le jeton d'info **VOUS ÊTES**, un pouce vers le haut, le jeton d'info **VOUS ÊTES**, puis le jeton du Snake Charmer. (*Cela montre qu'il est maintenant bon et le Snake Charmer.*) Rendormez le nouveau Snake Charmer.

Dans l'étrange situation où le Snake Charmer est méchant, ou le Démon bon, échangez leurs alignements comme approprié.

{{< /example >}}

{{< example >}}

{{< /example >}}

{{< example >}}
Le Pit-Hag se transforme lui-même en Snake Charmer. Ensuite, le Snake Charmer choisit un joueur qui est le Fang Gu. Le Snake Charmer devient le Fang Gu, tandis que le Fang Gu devient le Snake Charmer et est empoisonné. Les deux restent méchants.
{{< /example >}}
