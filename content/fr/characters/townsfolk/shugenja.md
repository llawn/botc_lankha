## Résumé

**Vous commencez la partie en sachant si votre joueur méchant le plus proche est dans le sens horaire ou antihoraire. Si équidistant, cette information est arbitraire.**

Le Shugenja fait confiance aux joueurs à sa gauche ou à sa droite.

* Le joueur méchant le plus proche est celui avec le plus petit nombre de pas du Shugenja à ce joueur méchant.
* Si les joueurs méchants sont « équidistants », cela signifie que le joueur méchant le plus proche dans le sens horaire est au même nombre de pas du Shugenja que le joueur méchant le plus proche dans le sens antihoraire.
* Si les joueurs méchants sont équidistants, le Conteur donne une information « arbitraire » au Shugenja. Cela signifie que le Conteur choisit de dire au Shugenja si le joueur méchant le plus proche est dans le sens horaire ou antihoraire.
* Le Shugenja ne sait pas si son information est arbitraire ou non.
* Le Shugenja n'apprend pas à combien de pas se trouve le joueur méchant.
* Si un Shugenja est créé en cours de partie, le Shugenja se réveille cette nuit-là pour recevoir son information.
* Le Conteur communique avec le Shugenja en pointant un doigt dans la direction appropriée, mais peut utiliser d'autres méthodes s'il le souhaite.

## Comment animer

Durant la première nuit, réveillez le Shugenja. Si le joueur méchant le plus proche est dans le sens horaire, pointez votre doigt horizontalement dans cette direction. Si le joueur méchant le plus proche est dans le sens antihoraire, pointez votre doigt horizontalement dans cette direction. Si les deux joueurs méchants les plus proches sont équidistants, pointez votre doigt horizontalement dans l'une ou l'autre direction. Rendormez le Shugenja.

## Citation

*これは夢。それも夢。すべて夢です。*
