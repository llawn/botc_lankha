## Résumé

**Vous et un joueur adverse vous connaissez mutuellement. Si le joueur bon est exécuté, le camp mauvais gagne. Le camp bon ne peut pas gagner si vous êtes tous les deux en vie.**

Le Jumeau Maléfique reflète un personnage bon, de sorte que les joueurs ne savent pas quel jumeau est bon et quel jumeau est mauvais.

* Le Jumeau Maléfique est lié à un joueur bon, choisi par le Conteur, appelé le Bon Jumeau.
* Lors de la première nuit, le Jumeau Maléfique et le Bon Jumeau se réveillent tous les deux, établissent un contact visuel et apprennent le personnage de l'autre.
* Si le Bon Jumeau est exécuté, le camp mauvais gagne. Si le Jumeau Maléfique est exécuté, la partie continue. Un Jumeau Maléfique mort n'a pas de capacité, donc le camp mauvais ne gagne pas si le Bon Jumeau est exécuté plus tard.
* Le camp bon ne peut pas gagner tant que les deux jumeaux sont en vie. Même si le Démon est tué, la partie continue. Le camp bon devra tuer le Jumeau Maléfique ainsi que le Démon pour gagner.
* Si un joueur bon est transformé en Jumeau Maléfique, il reste un joueur bon, un joueur mauvais devenant son jumeau. Peu importe quel jumeau est quel personnage, ce qui compte c'est leur alignement — le camp bon peut exécuter le joueur mauvais en toute sécurité, mais s'il exécute le joueur bon, le camp mauvais gagne.
* Si les deux jumeaux sont du même alignement, le Conteur choisit un nouveau jumeau.

## Comment animer

Lors de la mise en place de la partie, choisissez un Bon Jumeau en marquant n'importe quel personnage bon avec le rappel **JUMEAU**. Pendant la première nuit, réveillez les deux jumeaux. Laissez-les établir un contact visuel. Montrez le Jumeau Maléfique du doigt, puis montrez le jeton de Jumeau Maléfique au Bon Jumeau. Montrez le Bon Jumeau du doigt, puis montrez le jeton de personnage du Bon Jumeau au Jumeau Maléfique. Rendormez-les tous les deux.

Tant que les deux jumeaux sont en vie, le camp bon ne peut pas gagner. Si un jumeau d'alignement bon est exécuté, le camp mauvais gagne.

Si une Mégère des Fosses crée un Jumeau Maléfique pendant la dernière nuit, faites en sorte que l'autre jumeau soit le joueur Démon ou un joueur mort, afin que le camp bon puisse encore gagner.

Il est rarement judicieux de choisir un Voyageur comme jumeau.

## Exemples

{{< example >}}
Les deux jumeaux prétendent être l'Oracle. Le Jumeau Maléfique est exécuté. La partie continue.
{{< /example >}}

{{< example >}}
La Mégère des Fosses transforme le Sage, qui est aussi le Bon Jumeau, en Mutant. Les deux jumeaux essaient de convaincre le groupe qu'ils sont le Mutant. Le Conteur exécute immédiatement le Mutant, qui est aussi le Bon Jumeau. La partie se termine et le camp mauvais gagne.
{{< /example >}}

{{< example >}}
Le Bon Jumeau et le Jumeau Maléfique prétendent tous deux haut et fort être l'Artiste. Les deux joueurs s'approchent du Conteur pour poser une question en privé. Les joueurs bons, confus, exécutent le Démon. La partie continue, sans mort pendant la nuit désormais.
{{< /example >}}

{{< example >}}
La Mégère des Fosses transforme un joueur bon en Jumeau Maléfique, qui reste bon. Le groupe exécute le Jumeau Maléfique d'alignement bon. Le camp mauvais gagne.
{{< /example >}}
