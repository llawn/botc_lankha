## Résumé

**Si vous déclarez publiquement être le Gobelin lorsque vous êtes nommé et que vous êtes exécuté ce jour-là, votre équipe gagne.**

Le Gobelin se venge si la ville l'exécute en connaissance de cause.

* Si le Gobelin est exécuté, le camp mauvais gagne.
* ...mais pour que cela se produise, le Gobelin doit dire au groupe qu'il est le Gobelin lorsqu'il est nommé, mais avant que les votes n'aient lieu, et d'une manière que tout le monde entend. Les joueurs bons doivent connaître le risque.
* Si le Gobelin est exécuté sans avoir dit au groupe qu'il est le Gobelin lorsqu'il est nommé, le Gobelin meurt et la partie continue normalement.
* Le Gobelin doit avoir déclaré être le Gobelin aujourd'hui pour que sa capacité fonctionne. Le dire au groupe hier, ou même tous les jours précédents, ne compte pas.
* N'importe quel joueur peut déclarer être le Gobelin lorsqu'il est nommé.

## Comment animer

Si le Gobelin déclare être le Gobelin lorsqu'il est nommé (et avant que le vote ne commence), déclarez que ce joueur a déclaré être le Gobelin, afin que tout le groupe l'entende. Placez le rappel **A DÉCLARÉ** à côté du jeton de Gobelin.

Si le Gobelin est exécuté et qu'il est marqué **A DÉCLARÉ**, déclarez que le camp mauvais gagne.

Si un joueur qui n'est pas le Gobelin déclare être le Gobelin lorsqu'il est nommé, agissez comme s'il était le Gobelin. Déclarez au groupe qu'il a déclaré être le Gobelin et faites semblant de déplacer un jeton de rappel dans le Grimoire.

## Citation

*Vous ne voulez pas insulter les gobelins. Vraiment, vraiment pas. Et d'ailleurs... je peux reprendre un morceau de gâteau ?*
